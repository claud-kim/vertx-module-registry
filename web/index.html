Register your module

To register your own module, fill out the registration form.

Name
Please provide the identifier of your module so it can be verified. This will look like 
com.yourcompany~your-module-name~your-module-version
For example 
io.vertx~mod-mongo-persistor~2.0.0-beta2
Please note that we don't accept SNAPSHOT releases.
Location
  • For maven central, you don't need any additional information.
  • If you host your own maven repository, you need to provide the prefix URL for it. Maven central uses 
    http://repo1.maven.org/maven2/
    for example.
  • For bintray, you don't need any additional information.
  • If you host at your own webspace, you need to provide a complete URL to your ZIP-archive.
